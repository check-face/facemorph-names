Generated faces from baby names

Facemorph names lets you view generated images which are seeded from a list of popular baby names.

Click on a face to select it, then click on another to morph between them using facemorph.me

The baby names are based on data.gov.au male and female baby names in South Australia 2017.

See facemorph.me for information on how the faces are generated.

If the grid doesn't load, please check that you have javascript enabled.
